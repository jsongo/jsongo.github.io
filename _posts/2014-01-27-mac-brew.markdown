mac系统也可以像ubuntu里的apt-get一样安装系统软件包或库，brew。 
https:&#47;&#47;github.com&#47;Homebrew&#47;homebrew用git clone下来之后，安装，然后就可以用了： 
brew search [表达式] 
用来搜索想着的软件包，如brew search apache 
安装： 
brew install [软件名] 
卸载： 
brew uninstall [软件名]
